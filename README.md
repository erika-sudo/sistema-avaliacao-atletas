# 🏅 Sistema de Avaliação de Atletas

Sistema desenvolvido em JavaScript que calcula a média válida de atletas em competições de ginástica artística, descartando a maior e menor nota entre as 5 recebidas.

## 🚀 Tecnologias

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=for-the-badge&logo=nodedotjs&logoColor=white)

## ⚙️ Como executar

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado na sua máquina

### Passos

```bash
# Clone o repositório
git clone https://github.com/erika-sudo/sistema-avaliacao-atletas.git

# Entre na pasta do projeto
cd sistema-avaliacao-atletas

# Execute o arquivo principal
node notas-atletas.js
```

## 📋 Regras da competição

- Cada atleta recebe **5 notas**
- A **maior** e a **menor** nota são descartadas
- A média é calculada com as **3 notas restantes**

## 💻 Exemplo de saída

```
Atleta: Ana Lima
Notas: 8.5, 9.0, 7.5, 9.5, 8.0
Média válida: 8.50
```

## ⚙️ Funcionalidades

- Ordenação automática das notas
- Remoção da maior e menor nota
- Cálculo da média com base nas regras da competição

## 🎯 Objetivo

Projeto desenvolvido para praticar lógica de programação, manipulação de arrays e aplicação de regras de negócio em JavaScript.
